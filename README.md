## Mars Rover Test


#### Setup

```bash
bundle install
```

#### Testing

```
bundle exec rspec spec
```

###### With Guard

Running the tests continuously when code changes

```
bundle exec guard
```

#### Coverage

After running the tests

```
open coverage/index.html
```

#### Running

```
ruby main.rb < input
```

#### Changing the input

Edit the input file in the root of this repository
